# Project job4j_accidents

## Project description
In this app I want to create an accidents report system with role division: users and intendants. 

### Stack: 
- **Java 17**
- **Spring Boot 2.7.10**
- **PostgreSQL 14**
- **Spring (Data, Security)**
- **Hibernate**

### How to launch 
Launch Main.class /src/main/java/ru.job4.accidents

### Requirements: 
- **Java 17**
- **Maven 3.8**
- **PostgresSQL 14**
